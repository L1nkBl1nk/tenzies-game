🎲 Tenzies Game

This project is a Tenzies game built with Vite + React.

How it works:

Roll the dice by clicking a button.

Hold dice that you want to keep by clicking on them.

The goal is to get all dice to show the same number.

Track your rolls and try to win in as few moves as possible!

Technologies used:

Vite – Fast build tool

React – Front-end library for interactive UI
